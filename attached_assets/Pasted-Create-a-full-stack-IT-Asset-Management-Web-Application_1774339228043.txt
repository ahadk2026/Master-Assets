Create a full-stack IT Asset Management Web Application with enterprise-level features. Do NOT modify existing UI design or layout, only extend functionality.

## CORE MODULES

1. Asset Inventory Management:

* Manage all IT assets (laptops, desktops, etc.)
* Fields based on Excel:
  SN, Description, Category, Sub-category, Asset Number, Owner, Employee ID, DOJ, Custodian, Department, Email, Assigned Date, Location, OS, RAM, ROM, OS Version, Serial Number, Make & Model, Supplier, Invoice, Warranty, Remarks
* Status: Available / Assigned / In Service

2. Employee Management:

* Login using Employee ID
* Optional password (hashed using bcrypt)
* Track:

  * failed_attempts
  * is_locked
  * locked_at
  * is_active

3. Login Security:

* If failed login >= 3:

  * Lock account
  * Prevent login
* Only Admin can unlock
* Reset attempts on successful login

---

## ASSET ASSIGNMENT & ACKNOWLEDGMENT

4. Asset Assignment:

* Admin assigns assets to employees
* Track assignment history

5. Acknowledgment System:

* Employee logs in → sees "My Assets"
* Click "Acknowledge"
* Store:
  asset_id, employee_id, acknowledgment_date, status, remarks

6. PDF Acknowledgment:

* Generate PDF after acknowledgment
* Include:

  * Employee details
  * Asset details
  * Assigned date
  * Declaration text
  * Signature placeholder
* Allow:

  * Download
  * Print
* Store reference in database
